
# NE Retail Hub | Regional Logistics Grid

A professional inventory management and branch reordering platform for the North East regional network.

## Access Protocols

### Regional Administrators
- **Portal**: `/admin/login`
- **Credentials**: Any email containing `admin` (e.g., `admin@retail.com`) with a secure password.
- **Capabilities**: SKU provisioning, node authorization, global order tracking, and AI-driven inventory synthesis.

### Branch Managers
- **Portal**: `/register` (Compulsory Onboarding) or `/login`
- **Credentials**: Personal work email and secure passkey.
- **Capabilities**: Real-time stock reordering, order history tracking, and branch registry management.

## Sharing with Partners
This development workstation is private. To share the app with friends or mobile devices:
1. Initialize Firebase in your local terminal.
2. Run `firebase deploy`.
3. Share the generated public URL (e.g., `https://your-project.web.app`).
