# **App Name**: Retails stocks

## Core Features:

- User Registration: Allows new users to create an account securely using a username and password.
- Secure Login: Enables users to log into their account using their registered username and password.
- Biometric Authentication: Provides an alternative login method via mobile fingerprint for enhanced security and convenience on supported devices.
- Order Submission: Authenticated users can submit new orders.
- Order History: Users can view a personalized history of their previously placed orders.
- User Profile Management: Allows users to view and update their personal details and account settings.

## Style Guidelines:

- Primary brand color: A deep, professional blue (#2260B5) signifying reliability and efficiency.
- Background color: A very light, subtle blue-gray (#ECF0F5) providing a clean and calm canvas.
- Accent color: A vibrant, clear sky blue (#26CDF2) used for interactive elements and key calls to action like 'Register' or 'Order'.
- Headline and body font: 'Inter' (grotesque-style sans-serif) for its modern, neutral, and highly readable qualities across all text.
- Utilize clean, universally recognizable line icons for key actions and navigation, such as user profiles, login, and order processes, to ensure clarity and a streamlined visual experience.
- A minimalist and responsive layout that prioritizes clear forms and distinct steps for registration, login, and order submission, optimizing for mobile-first interactions.
- Subtle, functional animations for feedback during form submissions, loading states, and state transitions (e.g., successful login) to enhance user experience without distraction.
